# spring-security-6-jsp

## Requirements
- Apache Maven 3.9.6
- Docker Client 27.1.1-rd
- OpenJDK 17

## Quick Start
1. Start MySQL container:
```
docker compose -f compose.yml up
```

2. Build project:
```
mvn clean install
```

3. Start application:
```
java -jar .\target\task-manager.jar
```
